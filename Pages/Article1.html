Kebakaran

Fire

Kebakaran adalah api besar dan merusak yang mengancam kehidupan manusia, kehidupan hewan, kesehatan, dan / atau harta benda. Itu juga dapat digambarkan sebagai kobaran api atau hanya api. Kebakaran dapat dimulai secara tidak sengaja, disebabkan secara alami, atau dibuat dengan sengaja.
